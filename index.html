راهنما

راهنمای استفاده از ماشین حساب:

1. اعداد : از دکمه های 0 تا 9 برای استفاده از اعداد استفاده کنید

2. عملیات : "^" برای توان، x² برای مربع، x³ برای مکعب √ برای ریشه دوم log برای لگاریتم

3. توابع : sin، cos، tan برای محاسبات زاویه‌ای

4. π : مقدار عدد پی را وارد می‌کند

5. مبنا: برای محاسبه مبنا ابتدا عدد وارد کرده و سپس دکمه مبنا را بزنید مبنای قابل محاسبه در ماشین حساب مبنای (2-8-10)
